# LeadIQ — AI Lead Qualifier & Router

Short one paragraph description of what it does.

## Tech Stack
- Frontend: Vite + React + TypeScript + TailwindCSS
- Auth: Clerk (Google OAuth)
- Automation: n8n Cloud
- AI: OpenRouter
- Storage: Airtable
- Notifications: Slack
- Hosting: Netlify

## Features
- AI lead scoring (0-100)
- Automatic hot/warm/cold routing
- Google OAuth via Clerk
- Serverless webhook proxy (webhook URL never exposed to client)

## Setup
Just mention: clone repo, npm install, add VITE_CLERK_PUBLISHABLE_KEY to .env.local,
set WEBHOOK_URL in Netlify environment variables, deploy.
